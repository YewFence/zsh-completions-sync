## zcs generate

Generate completion scripts.

```
zcs generate [tool...] [flags]
```

### Options

```
  -h, --help            help for generate
  -j, --jobs int        Maximum number of tools to synchronize concurrently. (default 8)
  -o, --output string   Output directory for generated completion scripts.
  -s, --scope string    Generate completions for the selected scope. (default "global")
```
